Can I customize the dunning emails?
Yes — Starter and above include customizable templates with your logo, colors, and messaging. Pro adds full HTML customization.
How is this different from Stripe's built-in retries?
Stripe retries are basic and static. SimpleRecover uses adaptive logic (retry timing based on decline reason), multi-channel recovery (email + in-app), and gives you analytics Stripe doesn't provide.
Is my data secure?
Absolutely. We never store full card numbers. All data is encrypted at rest and in transit. SOC 2 Type II certified. We only need webhook events — no API keys with write access.
What happens if a payment can't be recovered?
We flag it for your team via Slack or email with the full context (customer history, decline reason, suggested action). Some payments need a human touch — we make sure you know when.
Can I cancel anytime?
Yes. No contracts, no setup fees. Cancel in your dashboard in 2 clicks. If you're unhappy, we'll refund your last month — no questions asked.

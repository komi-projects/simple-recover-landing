Gmail SMTP Setup

Configure Gmail to send dunning emails from SimpleRecover.

1

Enable 2-Factor Authentication

Go to Google Account → Security and enable 2-Step Verification. You must have 2FA enabled to use App Passwords.

2

Generate an App Password

Go to Google Account → App Passwords:

  • Select app: "Mail"
  • Select device: "Other (Custom name)"
  • Name it: "SimpleRecover"
  • Click "Generate"
  • Copy the 16-character password
3

Add to .env

Add these lines to your backend .env file:

SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_SECURE=false
SMTP_USER=komi54666@gmail.com
SMTP_PASS=your-app-password-here

⚠️ Never commit .env to git. It's already in .gitignore.

4

Test It

Run the test script:

cd simplerecover/backend
node test-smtp.js

If you see "✅ Email sent successfully", you're good to go.

5

Gmail Sending Limits

  • 500 emails/day for Gmail accounts
  • 100 recipients/email maximum
  • Rate limit: ~1 email/second recommended
  • For higher volume, upgrade to Google Workspace ($6/mo)
